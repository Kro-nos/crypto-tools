## To Run the app on local

From your terminal:

### Clone the repo

```sh
git clone https://github.com/shvam0000/crypto-tools
```

### Install all the dependencies

```sh
npm i / yarn install
```

### Run the app

```sh
npm run dev / yarn dev
```
